# AGENTS

This folder stores agent workflow artifacts for the workspace.

## Workflow

- Load `agent-issue-log.md` and review recent issue facts before starting work.
- Load `agent-guidelines.md` and follow the stable precautions distilled from repeated issues.
- Load `history.md` and follow the record rules.
