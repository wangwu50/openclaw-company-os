# 🏢 一人公司 OS

**基于 [OpenClaw](https://openclaw.ai) 的 AI 员工系统。** 一个人当 CEO，剩下的交给 AI 员工——产品、工程、运营、市场、财务全部到位，每天主动汇报、等你拍板。

---

## 功能

- **单一对话入口**：你只和 COO/总指挥 Jordan 对话，由其调度其他角色执行
- **6 位内置 AI 员工**：产品经理 Alex、COO/总监 Jordan、工程 Sam、运营 Maya、市场 Leo、财务 Chris
- **主动日报**：每位员工按固定时间自动汇报进展（工作日 9am–2pm 错开触发）
- **CEO 决策台账**：员工遇到分叉点自动提交 `[待决]` 请求，CEO 一键确认后员工立即执行
- **依赖式目标拆解**：目标会拆成带依赖关系的任务（非简单并行派发），按顺序推进
- **员工协作**：员工可主动查询同事动态，自动协调工作
- **目标工作台（隔离模式）**：按目标切换工作区，待决事项/动态/任务独立查看与处理
- **自定义员工**：用自然语言描述职位，AI 生成完整员工档案并立即上岗
- **Web UI**：内置管理界面，大厅总览 / 员工聊天 / 全员会议 / 决策台账

## 安装

### 前置条件

- [OpenClaw](https://openclaw.ai) 已安装并配置好 Anthropic API Key
- OpenClaw 版本 `>= 2026.3.14`
- Node.js `>= 22`（使用内置 `node:sqlite`，无需额外编译）

### 安装步骤

**第一步：克隆仓库**

```bash
git clone https://github.com/wangwu50/openclaw-company-os.git
```

**第二步：安装插件**

```bash
openclaw plugins install ./openclaw-company-os
```

安装完成后重启网关：

```bash
openclaw gateway restart
```

然后访问：

```
http://localhost:18789/company
```

> 如果网关开启了 token 认证，访问 `http://localhost:18789/company?token=YOUR_TOKEN`

## 使用

### 大厅（总览）

打开 `/company`，可以看到：
- 当前季度目标及任务进展
- 员工待 CEO 决策事项（点击确认即可通知员工继续执行）
- 最近员工报告

### 目标工作台（隔离）

打开 `/company/workspace`，可以按目标切换工作区：
- 只看该目标的待决事项
- 只看该目标的员工动态
- 只看该目标的任务看板

### 与员工聊天

默认只开放「总指挥」聊天入口。你向总指挥下达目标/问题，由总指挥协调其他员工执行并汇总反馈。

### 设置季度目标

在大厅点击「+ 添加目标」，AI 会自动将目标拆解为每位员工的具体任务，员工立即开始执行。

### 添加自定义员工

点击侧边栏「员工」旁的 `+` 按钮，用自然语言描述你想要的职位，AI 自动生成员工档案。
